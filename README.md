# Southeast Data Librarian Symposium 2022

Website for the 2022 Southeast Data Librarian Symposium (SEDLS). The SEDLS is intended to provide a low-cost opportunity for librarians and other research data specialists to gather and explore developments in the field of data librarianship, including the management and sharing of research data.

This is a static site built with hand-coded HTML, CSS, and a little javascript.  No external libraries are needed.

Questions about the contents can be sent to se.datalibrarain@gmail.com, questions about the code can be sent to gwenexner@gmail.com.
